Q＆A

  • 表示結果が明らかにおかしい。
    • htmlの<p>タグを認識して分析をしています。元の例会アルバムでタグの構造が普段と違うと、うまく分割ができずに、文章に不備が生じてしまいます。
    • 例 通し番号765「勝田さんの授業研究」は、元のサイトのpタグが閉じられていないので、うまく表示されません。
